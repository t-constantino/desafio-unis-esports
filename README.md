

# organização do evento

O evento ocorrerá dia 13 de junho de 2026, das 9:00 até as 12:00. A inscrição dos alunos será realizada pelo site unisesports.com.br e as equipes já devem ser previamente formadas, por conta própria dos mesmos. Os alunos deverão levar seus prórpios dispositivos mobile para jogar no campeonato.


# regulamento das competições

As competições serão realizadas entre 8 escolas de forma eliminatória, tendo uma equipe de 4 integrantes para representar cada escola. As equipes irão competir entre si até chegarem na fase final, onde as duas últimas equipes que sobreviverem competirão em uma partida final que decidirá a vencedora.

As partidas serão supervisionadas pela nossa equipe para garantir que ocorrerão de forma justa e respeitosa. Quaisquer falhas técnicas envolvendo a conexão Wi-Fi da faculdade ou os dispositivos dos alunos no meio do jogo resutará na desconsideração da partida, sendo recomeçada uma nova sempre que necessário.
