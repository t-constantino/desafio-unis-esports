DESAFIO UNIS eSports
📋 Sobre o Projeto
O UNIS eSports é um campeonato de esportes digitais desenvolvido pelo grupo de estudantes do curso Análise e Desenvolvimento de Sistemas (ADS) da Faculdade UNIS São Lourenço. O projeto integra tecnologia, educação e inclusão digital, promovendo competências essenciais como trabalho em equipe, raciocínio lógico, ética, cidadania digital e pensamento estratégico.


✅ Atividades Realizadas
1. Desenvolvimento da Plataforma Web
O grupo desenvolveu uma plataforma completa de inscrição e gerenciamento de participantes, incluindo:
Frontend

Interface responsiva e moderna para inscrição de participantes
Painel administrativo intuitivo para gerenciamento de dados
Design otimizado para experiência do usuário
Compatibilidade com múltiplos dispositivos

Backend

APIs para gerenciamento de inscrições
Sistema de autenticação e autorização
Integração com banco de dados para persistência de dados
Escalabilidade garantida para múltiplos usuários simultâneos

Sistema de Inscrição e Pontuação

Módulo completo de registro de participantes
Validação automática de dados
Relatórios em tempo real de rankings

Deploy

Colaboração com equipe de infraestrutura
Colocação da plataforma em produção
Garantia de estabilidade e segurança
Monitoramento contínuo do ambiente

Evidências:

Homepage do campeonato com informações principais
Painel administrativo funcional com métricas em tempo real
Formulário de inscrição com validação de dados
Sistema de gerenciamento de participantes operacional


2. Engajamento de Palestrantes
O grupo realizou ações de divulgação científica e conscientização, convidando especialistas para o seminário integrado ao evento.
Contato com Professores

3 professores convidados para ministrar palestras
2 confirmações recebidas
Aguardando retorno do terceiro palestrante

Temas Abordados
Tema 1: Hobby vs Vício

Identificação do ponto crítico entre hobby e comportamento viciante
Ênfase em eSports e jogos eletrônicos
Discussão sobre saúde mental e equilíbrio digital

Tema 2: Ergonomia em eSports

Melhores práticas de postura para jogadores
Arranjo ergonômico do espaço de trabalho
Prevenção de lesões e cuidados com a saúde

Status: Duas palestrantes confirmadas, seminário agendado para o dia do evento

3. Documentação para Patrocinadores
O grupo elaborou uma proposta profissional para captação de recursos, garantindo precisão e qualidade nas informações.
Processo de Desenvolvimento

Criação de documento inicial com proposta de patrocínio
Identificação e correção de erros críticos (datas, objetivos, dados quantitativos)
Validação de informações com a coordenação do projeto
Versão final com dados precisos e profissionais

Conteúdo da Proposta

Apresentação clara do projeto e seus objetivos
Dados estruturados do evento
Benefícios para patrocinadores
Público-alvo e alcance esperado

Evidências: Comunicações com patrocinadores em plataforma institucional, mostrando engajamento e feedback positivo
